# Korea Pro Center 🇰🇷

O'zbekistondagi #1 Koreya ta'lim konsalting markazi rasmiy veb-sayti.
Janubiy Koreya universitetlariga qabul, D-2 talaba vizasi, GKS grant va stipendiyalar.

**Live:** https://koreaproconsulting.netlify.app/

## ✨ Imkoniyatlar

- 🌐 **4 tilli** — O'zbek (default), Rus, Ingliz, Koreys (한국어)
- 🎓 **18+ universitet** — qidiruv, filtr va batafsil modal oynalar (Koreya, Yevropa, Yaponiya)
- 📰 **News bo'limi** — Google News'dan avtomatik Koreya ta'lim/grant/viza yangiliklari + curated zaxira
- 📨 **Lead forma** — EmailJS orqali (WhatsApp zaxira)
- 🗺️ **Google xarita** — Andijon ofis lokatsiyasi
- 📱 **To'liq responsive** — mobil, planshet, desktop
- 🔍 **SEO** — meta teglar, hreflang, JSON-LD, sitemap.xml, robots.txt
- ⚡ **Tez** — toza HTML/CSS/Vanilla JS, alohida optimallashtirilgan rasmlar

## 🛠 Texnologiya

Hech qanday framework yo'q — toza **HTML5 + CSS3 + Vanilla JavaScript**.
Tashqi: Google Fonts, Font Awesome, EmailJS.

## 📁 Tuzilma

```
index.html        — sahifa tuzilishi
style.css         — dizayn, animatsiyalar
script.js         — barcha funksiyalar
i18n.js           — 4 tilga tarjima
images_data.js    — viza rasmlari va logo (fayl yo'llari)
images/           — rasm fayllari
sitemap.xml       — SEO sitemap
robots.txt        — qidiruv robotlari
```

## 🚀 Lokal ishga tushirish

```bash
python -m http.server 8080
# http://localhost:8080
```
